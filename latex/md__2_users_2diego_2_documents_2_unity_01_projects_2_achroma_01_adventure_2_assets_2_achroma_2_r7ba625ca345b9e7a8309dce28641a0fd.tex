The goal of this tutorial is to get an in-\/game shop setup as quickly as possible by leveraging the pre-\/built shop popup in this example project. For a more detailed breakdown of how the shop popup was created, and steps on creating a similar shop popup of your own, please see the \href{https://github.com/Shopify/unity-buy-sdk-examples/blob/master/TUTORIAL.md}{\texttt{ full tutorial}}.\hypertarget{md__2_users_2diego_2_documents_2_unity_01_projects_2_achroma_01_adventure_2_assets_2_achroma_2_r7ba625ca345b9e7a8309dce28641a0fd_autotoc_md9}{}\doxysubsubsection{\texorpdfstring{Who this tutorial is for}{Who this tutorial is for}}\label{md__2_users_2diego_2_documents_2_unity_01_projects_2_achroma_01_adventure_2_assets_2_achroma_2_r7ba625ca345b9e7a8309dce28641a0fd_autotoc_md9}
This tutorial is aimed at any developer looking to quickly integrate the sale of physical merchandise into their game. Since this tutorial uses the pre-\/built shop popup, only basic Unity skills are required in order to get setup.\hypertarget{md__2_users_2diego_2_documents_2_unity_01_projects_2_achroma_01_adventure_2_assets_2_achroma_2_r7ba625ca345b9e7a8309dce28641a0fd_autotoc_md10}{}\doxysubsection{\texorpdfstring{Integrating the shop popup}{Integrating the shop popup}}\label{md__2_users_2diego_2_documents_2_unity_01_projects_2_achroma_01_adventure_2_assets_2_achroma_2_r7ba625ca345b9e7a8309dce28641a0fd_autotoc_md10}

We\textquotesingle{}ll start with a blank, default Unity project, as seen above.

Our first step is to add in the shop popup unitypackage, which can be downloaded \href{https://github.com/Shopify/unity-buy-sdk-examples/blob/master/UnityBuySDKShopPopup.unitypackage}{\texttt{ here}}. Drag this into the Assets area of your project.



When prompted, import the package contents.\hypertarget{md__2_users_2diego_2_documents_2_unity_01_projects_2_achroma_01_adventure_2_assets_2_achroma_2_r7ba625ca345b9e7a8309dce28641a0fd_autotoc_md12}{}\doxysubsubsection{\texorpdfstring{Add an instance of the Shop Popup prefab}{Add an instance of the Shop Popup prefab}}\label{md__2_users_2diego_2_documents_2_unity_01_projects_2_achroma_01_adventure_2_assets_2_achroma_2_r7ba625ca345b9e7a8309dce28641a0fd_autotoc_md12}


Next, go into the Prefabs folder, and drag an instance of the Shop Popup prefab into the scene.

In the options for the Shop Popup prefab, we can set our Access Token and Shop Domain. For more information on getting the Access Token, see \href{[https://help.shopify.com/api/storefront-api/getting-started\#obtaining-a-storefront-access-token}{\texttt{ Obtaining a storefront Access Token}}.\hypertarget{md__2_users_2diego_2_documents_2_unity_01_projects_2_achroma_01_adventure_2_assets_2_achroma_2_r7ba625ca345b9e7a8309dce28641a0fd_autotoc_md13}{}\doxysubsubsection{\texorpdfstring{Add the UI eventsystem}{Add the UI eventsystem}}\label{md__2_users_2diego_2_documents_2_unity_01_projects_2_achroma_01_adventure_2_assets_2_achroma_2_r7ba625ca345b9e7a8309dce28641a0fd_autotoc_md13}


In order for the UI events to fire, we must add in a UI \texorpdfstring{$>$}{>} Eventsystem game object.\hypertarget{md__2_users_2diego_2_documents_2_unity_01_projects_2_achroma_01_adventure_2_assets_2_achroma_2_r7ba625ca345b9e7a8309dce28641a0fd_autotoc_md14}{}\doxysubsubsection{\texorpdfstring{Test the shop popup}{Test the shop popup}}\label{md__2_users_2diego_2_documents_2_unity_01_projects_2_achroma_01_adventure_2_assets_2_achroma_2_r7ba625ca345b9e7a8309dce28641a0fd_autotoc_md14}


With these steps completed, the shop popup should be ready to use. There is a hidden button which triggers the shop popup, click in the middle of the view and the shop popup should appear.
