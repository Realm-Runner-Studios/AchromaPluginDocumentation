The location of the image as a URL. 

\begin{DoxyRefDesc}{Deprecated}
\item[\mbox{\hyperlink{deprecated__deprecated000091}{Deprecated}}]Previously an image had a single {\ttfamily src} field. This could either return the original image location or a URL that contained transformations such as sizing or scale.\end{DoxyRefDesc}


These transformations were specified by arguments on the parent field.

Now an image has two distinct URL fields\+: {\ttfamily original\+Src} and {\ttfamily transformed\+Src}.


\begin{DoxyItemize}
\item {\ttfamily original\+Src} -\/ the original unmodified image URL
\item {\ttfamily transformed\+Src} -\/ the image URL with the specified transformations included
\end{DoxyItemize}

To migrate to the new fields, image transformations should be moved from the parent field to {\ttfamily transformed\+Src}.
