This repo contains an example of how to use the Shopify SDK for Unity (\href{https://github.com/Shopify/unity-buy-sdk}{\texttt{ https\+://github.\+com/\+Shopify/unity-\/buy-\/sdk}}). The SDK allows Unity developers to sell products from their Shopify store (\href{https://www.shopify.ca/}{\texttt{ https\+://www.\+shopify.\+ca/}}) directly within games and other interactive experiences made within Unity.\hypertarget{md__2_users_2diego_2_documents_2_unity_01_projects_2_achroma_01_adventure_2_assets_2_achroma_2_rd81e2eb99022bff3afc81983047d5872_autotoc_md16}{}\doxysubsection{\texorpdfstring{Getting started}{Getting started}}\label{md__2_users_2diego_2_documents_2_unity_01_projects_2_achroma_01_adventure_2_assets_2_achroma_2_rd81e2eb99022bff3afc81983047d5872_autotoc_md16}
To get started, either checkout this project using Git, or download as a zip. The project can be opened in the Unity editor, a free trial version is available here \href{https://store.unity.com/download?ref=personal}{\texttt{ https\+://store.\+unity.\+com/download?ref=personal}}. The Shopify SDK for Unity functionality is integrated as a shop within a pop up menu. The Shopify SDK for Unity has already been included into this example, but in general, an integration is started by downloading and including the \href{https://github.com/Shopify/unity-buy-sdk/blob/master/shopify-buy.unitypackage}{\texttt{ https\+://github.\+com/\+Shopify/unity-\/buy-\/sdk/blob/master/shopify-\/buy.\+unitypackage}} into the target project.\hypertarget{md__2_users_2diego_2_documents_2_unity_01_projects_2_achroma_01_adventure_2_assets_2_achroma_2_rd81e2eb99022bff3afc81983047d5872_autotoc_md17}{}\doxysubsection{\texorpdfstring{Quickstart tutorial}{Quickstart tutorial}}\label{md__2_users_2diego_2_documents_2_unity_01_projects_2_achroma_01_adventure_2_assets_2_achroma_2_rd81e2eb99022bff3afc81983047d5872_autotoc_md17}
The example is comprised of a shop popup menu which demonstrates the usage of the Shopify SDK for Unity. While it is entirely possible to do a complete integration from scratch, the shop popup from this example can also be directly included into your game, and later tweaked to match a visual style. For information on how to get the example shop popup integrated into a game in a few steps, please see the \href{https://github.com/Shopify/unity-buy-sdk-examples/blob/master/QUICKSTART_TUTORIAL.md}{\texttt{ quickstart tutorial}}.\hypertarget{md__2_users_2diego_2_documents_2_unity_01_projects_2_achroma_01_adventure_2_assets_2_achroma_2_rd81e2eb99022bff3afc81983047d5872_autotoc_md18}{}\doxysubsection{\texorpdfstring{Tutorial}{Tutorial}}\label{md__2_users_2diego_2_documents_2_unity_01_projects_2_achroma_01_adventure_2_assets_2_achroma_2_rd81e2eb99022bff3afc81983047d5872_autotoc_md18}
For more details on how the example was created, and detailed steps that can be followed for integrating the Shopify SDK for Unity into other games, please see the \href{https://github.com/Shopify/unity-buy-sdk-examples/blob/master/TUTORIAL.md}{\texttt{ full tutorial}}.
