\doxysubsection{Detailed Description}
\doxylink{namespace_google}{Google} sign in API. 

This class implements the Google\+Sign\+In\+API for Unity. Typical usage is to set the Configuration options as needed, then get the Default\+Instance and call sign\+In or sign\+In\+Silently. See the \href{https://developers.google.com/identity}{\texttt{ Google Sign-\/\+In API documentation for more details.}}
